Set eyes red

El Set Eyes consta de una pulsera ancha en conjunto con dos angostas y un semanario.
  • Las pulseras están confeccionadas con cuentas de cristal de murano.
  • Las cuentas del semanario son también de cristal.
  • Todo el set tiene cierres corredizon individuales. Esto permite en primer lugar ajustar el tamaño a cualquier tipo de muñeca. Y en segundo lugar hace muy facil la colocación por uno mismo, para no tener que pedir ayuda!
